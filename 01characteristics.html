Characteristics of web typography

Typography is visual semantics—the use of visual encodings to convey structure and meaning.

01. Good web typography?

The visual display of text depends on many variables, such as user settings, context of use, device used, and size of the viewport. Good web typography adapts gracefully to these different contexts.

02. Characteristics

  1. Describe related and information hierarchy.

  2. Emphasize individual elements.

  3. Using CSS(Cascading Style Sheets) to reconcile the division between structural logic and visual logic.
